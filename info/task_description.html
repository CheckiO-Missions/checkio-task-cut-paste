Cut & Paste

There are a few ribbons that were glued from pieces of two colors.
You must form them into a single stripe by scissors and glue, as shown in the figure.
Permitted operations - cut up, glue, reverse.
Number of cuts should be minimized.

Input: list of strings, each string - "ribbon" with '1' as light piese else '0'.

Input for case on figure: 

data = ['11000101', '101']

Output: Python expression as a string, which returns the stripe.

Stripe - a string of alternating '0' and '1'.

Example output: 

"""
''.join((data[0][1:3], data[0][:1], data[0][3:4], data[0][-1:-5:-1], data[1]))
"""

Be care: the strings below would be rejected because they contradicts to the task. 

"'10101010101'"
"""data[0][1:4] * 4 + data[1]"""

How it is used:

If you'll solve this task, you can append to your resume:
"experience in metaprogramming" ;)

Precondition:

  • Total light and total dark segments differs not more than 1.
