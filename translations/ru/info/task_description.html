Cut & Paste

Есть несколько отрезков ленты, склееных из кусочков двух цветов.
Их нужно разрезать и склеить так, как показано на рисунке.
Допустимые операции - разрезать, склеить, перевернуть.
Число разрезов должно быть минимальным.

Входные данные: Список "отрезков ленты", каждый отрезок - строка, "1" if светлый else "0".

Входные данные в примере: 

data = ['11000101', '101']

Выходные данные: Выражение на Питоне, возвращающее искомую последовательность, как str.

Искомая последовательность - строка с чередующимися '0' и '1'.

Пример выходных данных: 

"""
''.join(data[0][1:3], data[0][:1], data[0][3:4], data[0][-1:-5:-1], data[1])
"""

Внимание: следующие варианты ответа будет отвергнуты, как не отвечающие условию задачи: 

"'10101010101'"
"""data[0][1:4] * 4 + data[1]"""

Зачем это нужно:

Решив эту задачу, вы смело можете добавить в своё резюме: "Владею навыками метапрограммирования" ;)

Предусловия:

  • Общие количества светлых и тёмных сегментов отличаются не более чем на 1.
